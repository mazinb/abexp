A/B Testing Tool Demo

Welcome to our A/B Testing Tool demo! This page showcases the functionality of our A/B testing tool, which allows you to perform experiments to compare different versions of a web page or application.

How It Works

Our A/B testing tool randomly assigns users to different variations (A or B) of an experiment. You can see which variation you are currently assigned to at the top of the page. Click the "Change Experience" button to switch between variations.

Click the "See API response" button to view the API response for the allocation of users to experiment variations. You can also navigate to the Events page to view a summary of tracked events and the Data page to explore experiment data.
